Agentic command center for Git worktrees.

## Installation

```bash
brew tap work3s/agentic-command-center
brew install --cask work3s
```

## Updating

```bash
brew upgrade --cask work3s
```

## Uninstalling

```bash
brew uninstall --cask work3s
```
